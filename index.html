Projects

All project are either forked and initiated by me on github

Intelligent Hat

Creation of a hat equipped with four ultrasonic sensors and four vibrating motors, positioned on each side of the hat, to help blind people detect and distinguish obstacles.

Egypt

Development of a multiplayer game based on protecting the Egyptian pyramids from aliens, with a client-server architecture implemented using threads to ensure a real-time experience while optimizing interference management. This game also includes a chatbox to allow players to collaborate more effectively.

Jumper

Development of a simulation of an agent in the form of a cube that evolves to reach the maximum level, by implementing reinforcement learning and using the torch.nn library from Pytorch for neural network implementation. Addition of short-term and long-term memory algorithms to enhance the simulation.

ChatLAN

Developed a feature-rich Android chat application enabling communication through device contacts. Implemented robust server-client architecture with secure authentication protocols. Utilized SQLite for efficient local data management and offline functionality. Delivered a seamless user experience with contact-based chat integration and real-time messaging.

Portadox

An adventure game inside my brain, Utilized pure Java and created an engine with java to fast scan maps and to optimize complexity .
